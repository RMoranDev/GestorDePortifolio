# Gestor de Projetos - Python

Este é um projeto acadêmico desenvolvido como parte dos meus estudos em **Análise e Desenvolvimento de Sistemas (PUCPR)**.

## 🚀 Objetivo
O sistema permite gerenciar tarefas e projetos de forma simples via terminal, utilizando conceitos fundamentais de Python como loops `while True`, estruturas de decisão e manipulação de listas.

## 🛠️ Tecnologias
- **Linguagem:** Python 3.14
- **IDE:** PyCharm
- **Versionamento:** Git & GitHub
